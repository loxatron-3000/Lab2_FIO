# Лабораторная работа №2 - Юнит-тестирование

## Задание
Написать юнит-тесты для двух методов . Минимум 20 тестов на каждый метод. Найти ошибки в чужом коде. Оформить отчет. Залить на GitHub.

## Что сделано
- Свой метод (OrderCalculator) - 24 теста, все проходят
- Чужой метод (BankAccount) - 22 теста, 11 проходят, 11 не проходят

## Найденные ошибки в чужом коде
1. Можно пополнить счет на отрицательную сумму
2. Можно снять отрицательную сумму
3. Можно снять ноль рублей
4. Счет блокируется при нулевом балансе (нужно при отрицательном)
5. Нет проверки блокировки при пополнении
6. Нет проверки блокировки при снятии
7. Можно перевести деньги самому себе
8. Нет проверки на null при переводе
9. Неправильная формула процентов
10. Можно разблокировать счет с недостаточным депозитом
11. Начисление процентов на заблокированный счет
